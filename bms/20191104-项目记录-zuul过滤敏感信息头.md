## 20191104 - 项目记录 - Zuul 过滤敏感信息头

- 描述：httpServletRequest.getHeader 一直获取不到值

~~~java
String uid = request.getHeader("X-SSO-ACCOUNT");
// 打印一直为 null
System.out.println("uid:--------------" + uid);
~~~

- 原因：Zuul 会对敏感信息头做过滤
